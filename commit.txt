1) Зайти в репозиторий:
cd /c/REPOSITORY/ITMO.Course2021.HW

2) Приверить изменения:
git status

3) Добавить папку или файл:
git add /c/REPOSITORY/ITMO.Course2021.HW

4) Оставить коммит об изменнии:
git commit -m "CREATE labs HTML5-CSS3-JS"
git commit -m "add index.html"
git commit -m "правка картинки"
git commit -m "readme.md"

5) Отправить все на сервер:
git push
